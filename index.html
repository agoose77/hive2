Hive2

A Visual Programming System

View on GitHub Download .zip Download .tar.gz

Features

  • Create reusable Python components with visual editor, or Python code
  • Generate UI interface from Python node definition
  • Real-time visual debugging in-editor
  • Extensive standard library for common data operations
  • Build time connectivity type checking

History

Hive2 is a system for creating and connecting reusable Python components. It provides a Qt editor to define connectivity without writing a line of code. Based upon the hivesystem module, Hive2 focuses upon run-time performance; building connectivity at build time.
